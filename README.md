# EcoCity
"مشروع EcoCity هو موقع ويب تفاعلي صُمم بـ HTML/CSS/JS لتعزيز الوعي البيئي تماشياً مع رؤية مصر 2030. يستعرض الموقع حلول التغير المناخي وإنجازات الدولة كالعاصمة الإدارية، متميزاً بتصميم متجاوب يدمج بين الأزرق التقني والأخضر البيئي، مع واجهة مستخدم بسيطة وشاشات أهداف واضحة، لتقديم تجربة تعليمية رقمية مبتكرة وسهلة التشغيل."
